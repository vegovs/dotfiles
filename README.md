# Dotfiles 

Some config files for: 
* Git
* Vim
* bash
* tmux

setup.sh:
* Installs: 
	* Vim
		* Vundle 
	* Tmux 
* Symlinks:
	* .bashrc 
	* .gitconfig 
	* .vimrc 
	* .tmux.config
	* .gitignore_global  
	* OneDrive
	* GoogleDrive
	* Dropbox
* Configures:
	* Git global .gitignore

## Usage
```
git clone https://github.com/vegovs/dotfiles.Git
cd dotfiles
bash setup.sh
```
