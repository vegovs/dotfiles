# Dotfiles 

Some config files for: 
* Git
* Vim
* bash
* tmux

setup.sh:
* Installs: 
	* Pandoc
	* Vim
	* Tmux 
	* Vundle 
* Symlinks:
	* .bashrc 
	* .gitconfig 
	* .vimrc 
	* .tmux.config
	* .gitignore_global  
	* OneDrive
	* GoogleDrive
	* Dropbox
* Configures:
	* Git global .gitignore

## Usage
```
git clone https://github.com/vegovs/dotfiles.git && cd dotfiles && sudo bash setup.sh
```
